If you want to build our source code into a viewable website, please follow these instructions:

1. Install Node.js (version 20+). You can check if you have node in the correct version by typing "node -v" in your terminal.
2. Change the directory into the one containing the project (home to the index.html file).
3. Install vite using "npm i vite".
4. Install all the necessary packages using "npm i".
5. Run the development server with "npm run dev" and open localhost at the port returned by vite.
